# Correlation-analysis-centromere
Correlation analysis for centromeric proteins based on pan-cancer datasets. The code is accompanying the manuscript "CENP-C is guilty by association" by Kochanova N.Y., Samejima I. and Earnshaw W.C.

The two versions of the manuscript are available here:
https://rdcu.be/eU7XP
https://rdcu.be/eU7Ye

Kochanova, N.Y., Samejima, I. & Earnshaw, W.C. Sustainable integrative cell biology: CENP-C is guilty by association. Chromosome Res 33, 27 (2025). https://doi.org/10.1007/s10577-025-09784-0

Kochanova, N.Y., Samejima, I. & Earnshaw, W.C. Sustainable integrative cell biology: CENP-C is guilty by association. Chromosoma 134, 12 (2025). https://doi.org/10.1007/s00412-025-00840-6
